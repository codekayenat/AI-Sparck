A simple learning platform for students to learn French through short tutorials, exercises, and fun facts.

## Project structure
- server.js - Node + Express server serving static files and API endpoints
- public/ - frontend (HTML, JS, CSS)
- data/ - sample tutorials, exercises and fun facts JSON
